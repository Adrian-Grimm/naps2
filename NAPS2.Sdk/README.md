# NAPS2.Sdk

NAPS2.Sdk is a fully-featured scanning library, supporting WIA, TWAIN, SANE, and ESCL scanners on Windows, Mac, and Linux.

|           | Windows | Mac | Linux |
|-----------|---------|-----|-------|
| **WIA**   | X       |     |       |
| **TWAIN** | X       | X   |       |
| **SANE**  |         |     | X     |
| **ESCL**  | X       | X   | X     |

See the [Samples](https://github.com/cyanfish/naps2/tree/master/NAPS2.Sdk.Samples).
